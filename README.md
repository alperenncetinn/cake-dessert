## 📌 Proje Hakkında

Bu proje, **Emine Cake Dessert** için özel olarak tasarlanmış bir butik pasta web sitesidir. Müşteri talebi doğrultusunda **GitHub Pages** üzerinde yayınlanmak amacıyla bu depoda barındırılmaktadır.

### ✨ Özellikler

- 🎂 **Pasta Yapılış Animasyonlu Loading Ekranı** — 6 saniyelik etkileyici açılış
- 📱 **Tam Mobil Uyumlu** — Tüm cihazlarda kusursuz görünüm
- 📸 **Instagram Entegrasyonu** — Gerçek Instagram postları iframe ile gömülü
- 🎨 **Modern & Premium Tasarım** — Glassmorphism, gradient'ler, micro-animasyonlar
- ⚡ **Hızlı & Hafif** — Framework kullanılmadan saf HTML/CSS/JS ile geliştirildi

### 🛠️ Teknolojiler

| Teknoloji | Kullanım |
|-----------|----------|
| HTML5 | Sayfa yapısı |
| CSS3 | Stil, animasyonlar, responsive tasarım |
| JavaScript | Loading ekranı, scroll animasyonları, menü |
| Google Fonts | Playfair Display, Poppins, Great Vibes |
| Font Awesome | İkonlar |
